# Hugolify demo multisite

## Sites

- site A
- site B

## Commands

Install packages

```
yarn
```

### Site A

Watch

```
yarn watch-site_a
```

Build

```
yarn build-site_a
```

### Site B

Watch

```
yarn watch-site_b
```

Build

```
yarn build-site_b
```
